# Making calls to the library{#making-calls-to-the-library}

スクリプトからプラグインを呼び出す場合は、名前空間を読み込みます。

* **C#：**&#x200B;次を使用します。`com.adobe.mobile;`

名前空間を読み込んだ後、ADBMobileクラスの静的メソッドを介してプラグインを直接呼び出すことができます。
