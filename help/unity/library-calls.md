# ライブラリへの呼び出し{#making-calls-to-the-library}

スクリプトからプラグインを呼び出す場合は、次の名前空間を読み込みます。

* **C#:** using `com.adobe.mobile;`

名前空間を読み込んだ後、ADBMobileクラスの静的メソッドを使用して、プラグインを直接呼び出すことができます。
